# Agent Chat Workflow

The assistant is strongest when it can move from retrieval to a safe proposal:

1. read the source note
2. pull nearby notes for context
3. suggest a summary, tag change, or edit
4. wait for a human accept or reject decision

Working rules:

- keep prompts grounded in concrete notes, not vague topics
- prefer proposal cards over direct mutation
- use destructive actions only when a second confirmation makes sense
