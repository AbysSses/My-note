# Sidebar Information Architecture

The left rail should answer three questions fast:

- what notes matter right now
- what tasks need action
- which projects are active, paused, done, or archived

Design constraints:

- keep buckets stable so memory builds over time
- collapse finished work by default
- make the current project obvious without adding noise
